SU-10A相关

1、默认固件

  默认的固件是测试语音识别灵敏度的，没有添加IO输出功能，如果需要IO输出功能，可以在平台上面自定义生成，模块可以发出IO电平信号和串口数据，但是接收串口数据和检测输入IO信号需要二次开发(具体可询问客服)

2、10a支持哪些方式烧录？

  先用烧录器烧录一个固件后，才可以串口烧录

3、有控制IO吗？

  没有

4、语音唤醒词能自定义吗？

  自定义要购买我们的烧录器才行
